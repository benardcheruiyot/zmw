# Client App

Vite + React frontend for the MoMo instant loan application flow.

## Current Flow Settings

- Brand: MoMo
- Country: Zambia
- Mobile prefix: `+260` (9-digit local input)
- Currency label: ZMW
- Loan amount range: `ZMW 5,000` to `ZMW 45,000`

## Development

```bash
npm install
npm run dev
```

Default dev URL: `http://localhost:5173`

## Build

```bash
npm run build
```
